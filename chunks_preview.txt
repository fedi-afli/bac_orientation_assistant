# SECTION 4: MATHEMATICAL FORMULAS FOR SCORING & MERIT RANKS
The academic rank is based on a dual-metric calculation: the Global Formula (Formule Globale - FG) and the program-specific Orientation Score (T).
## 4.1: The Global Formula (FG)
The FG establishes the foundational grade weight out of 100 based on core terminal examination grades.
- For General Baccalaureate (Bac Général):
Formula: FG = (4 * MF) + (1.5 * PH) + (0.5 * HG) + (0.5 * E.Sc) + (1.5 * GO) + (1 * F) + (1 * LV)
- For Technological Baccalaureate (Bac STMG):
Formula: FG = (4 * MF) + (1 * PH) + (1 * HG) + (1 * M) + (1 * GO) + (1 * F) + (1 * LV)
Abbreviation Glossary:
* MF: Moyenne Finale (Overall Final Baccalaureate Average Grade out of 20)
* PH: Philosophie (Philosophy Grade)
* HG: Histoire-Géographie (History-Geography Grade)
* E.Sc: Enseignement Scientifique (Scientific Education Grade - For General Baccalaureate only)
* M: Mathématiques (Mathematics Grade - For Technological Baccalaureate only)

## 4.2: Program Orientation Score (T)
The system calculates a custom score for every unique course preference code selected by the student.
- Base Rule: T = FG + (Moyenne des notes des matières spécifiques à la filière)
- Scope Limitation: Only specialty course tracks studied during the final terminal year (classe terminale) can be pulled into the specialized grade average component of the T score.
- Availability: Custom calculated FG grades and individual merit brackets are displayed inside www.orientation.tn starting July 16, 2025.

## 4.3: Adjustments for Remedial Session (Session de Rattrapage) Candidates
For high school seniors passing their baccalaureate via the second-chance session, individual grades and averages undergo standard arithmetic smoothing:
- Normalized Subject Grade = [(Grade from Main Session * 2) + Grade from Remedial Session] / 3
- Normalized Session Average = [(Average from Main Session * 2) + Average from Remedial Session] / 3
